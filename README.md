Prompts
===============================

version number: 0.0.1
author: pandada8

Overview
--------

Python package for cli prompts, inspired by Inquirer.js

Installation / Usage
--------------------

To install use pip:

    $ pip install prompts


Or clone the repo:

    $ git clone https://github.com/pandada8/prompts.git
    $ python setup.py install
